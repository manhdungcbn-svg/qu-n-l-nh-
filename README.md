# Dormitory Management System

## Overview
The Dormitory Management System is a web-based application designed to facilitate the management of dormitory rooms, tenants, contracts, and payments. This system provides an intuitive interface for administrators to efficiently handle various aspects of dormitory management.

## Project Structure
```
dormitory-management-system
├── index.html          # Main entry point for the application
├── css
│   └── styles.css     # Styles for the application
├── js
│   ├── app.js         # Initializes the application and manages navigation
│   ├── rooms.js       # Handles room management functionality
│   ├── tenants.js     # Manages tenant-related operations
│   ├── contracts.js    # Deals with contract management
│   ├── payments.js    # Manages payment records
│   └── reports.js     # Generates reports based on collected data
└── README.md          # Documentation for the project
```

## Features
- **Room Management**: Add, edit, delete, and view room details.
- **Tenant Management**: Add, edit, delete, and search for tenant information.
- **Contract Management**: Create, edit, and delete contracts between tenants and rooms.
- **Payment Management**: Record and manage payments, including calculating totals and updating payment statuses.
- **Reporting**: Generate reports for statistical insights on room occupancy, tenant information, and payment history.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the `index.html` file in a web browser to access the application.
3. Ensure that your browser supports local storage for data persistence.

## Usage Guidelines
- Navigate through the application using the menu links provided in the header.
- Fill out the forms to manage rooms, tenants, contracts, and payments.
- Use the search functionality to quickly find tenant information.
- Generate reports to view summaries of the dormitory's status.
